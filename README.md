## Descripción

El sistema permite la compra y seleccion de productos, de la misma forma que permite al usuario registrar los productos que desea comprar, así como el pago de los productos y tambien terminar el ciclo cuando quiera.

## objetivo

El objetivo de este proyecto es crear un sistema basico en python para gestionar compras, donde se pongan en practica conceptos como lista, ciclo y condicionales.Ademas, se busca aprender a trabajar en equipo utilizando herramientas como Git y GitHub para el control de verciones
